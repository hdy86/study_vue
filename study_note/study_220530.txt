* 프록시(proxy) 서버
  클라이언트가 자신을 통해서 다른 네트워크 서비스에 간접적으로 접속할 수 있게 해주는 응용 프로그램

* CORS
  웹 어플리케이션에서 도메인, 포트, 프로토콜이 다른 곳으로 HTTP 요청을 보낼 수 없도록 브라우저가 요청을 막는 보안 정책

* Vuex
  Vue.js 어플리케이션을 이한 상태관리 패턴+라이브러리
  모든 컴포넌트에 대한 중앙집중식 저장소 역할을 하며, 예측 가능한 방식으로 상태 변경 가능
  1. State : 프로젝트 전체에서 공통으로 사용할 변수 정의
  2. Getters : 가져올 수 있는 계산된 속성들
  3. Mutations : state 변경 가능
  4. Actions : Mutations 에 정의된 함수 실행

* Vuex 설치
  VS Code 터미널에 npm install vuex@next --save 로 입력

* 프로젝트 배포하기
  VS Code 터미널에 npm run build 입력하면 해당 프로젝트 내의 dist 폴더에 파일들이 만들어짐
